# Bastion: Typed Middleware Pipeline

**Status**: Draft | **Version**: 0.1 | **Part of**: [Bastion Design Spec](README.md)

---

## Core Concept

Middleware in Bastion transforms a `Conn` from one type state to another. The type parameter tracks what has been applied:

```march
# Type definitions for conn states
type Raw = {}
type Parsed = { body: Body }
type WithSession = { body: Body, session: Session }
type Authenticated = { body: Body, session: Session, current_user: User }
type WithDB = { db: Depot.Pool }

# A middleware is a function that transforms the conn's type state
type Middleware(a, b) = fn(Conn(a)) -> Conn(b)
```

---

## Built-in Middleware

```march
mod Bastion.Middleware do
  # Parses request body (JSON, form data, multipart)
  fn parse_body(conn: Conn(Raw)) -> Conn(Parsed) do
    body = case get_req_header(conn, "content-type") do
      "application/json" -> JSON.decode(conn.raw_body)
      "application/x-www-form-urlencoded" -> Form.decode(conn.raw_body)
      "multipart/form-data" -> Multipart.parse(conn.raw_body)
      _ -> %{}
    end
    %{conn | assigns: Map.put(conn.assigns, :body, body)}
  end

  # Loads session from signed cookie
  fn load_session(conn: Conn(Parsed)) -> Conn(WithSession) do
    session = conn
    |> get_req_cookie("_session")
    |> Session.verify_and_decode(conn.secret_key_base)
    %{conn | assigns: Map.put(conn.assigns, :session, session)}
  end

  # Requires authentication — returns error or enriched conn
  fn require_auth(conn: Conn(WithSession)) -> Result(Conn(Authenticated), Conn(WithSession)) do
    case conn.assigns.session.user_id do
      Some(user_id) ->
        user = MyApp.Users.get(user_id)
        Ok(%{conn | assigns: Map.put(conn.assigns, :current_user, user)})
      None ->
        Error(conn |> send_resp(401, "Unauthorized"))
    end
  end

  # Attaches a Depot connection pool to the conn
  fn with_db(conn: Conn(a), pool: Depot.Pool) -> Conn(a & WithDB) do
    %{conn | assigns: Map.put(conn.assigns, :db, pool)}
  end
end
```

## Pipeline Composition

Middleware is composed using March's pipe operator. The type system enforces correct ordering:

```march
mod MyApp.Endpoint do
  import Bastion.Middleware

  fn call(raw_conn: Conn(Raw)) do
    raw_conn
    |> parse_body()           # Conn(Raw) → Conn(Parsed)
    |> load_session()         # Conn(Parsed) → Conn(WithSession)
    |> with_db(MyApp.Pool)    # Conn(WithSession) → Conn(WithSession & WithDB)
    |> MyApp.Router.route()   # dispatch to handler
  end
end
```

Attempting to call `require_auth` before `load_session` would be a **compile-time type error** — the compiler knows `require_auth` expects `Conn(WithSession)` and won't accept `Conn(Parsed)`.

## Scoped Pipelines

Different route groups can apply different middleware stacks:

```march
mod MyApp.Router do
  import Bastion.Middleware

  # Public pages — no auth required
  fn route(conn: Conn(WithSession), :get, ["login"]) do
    MyApp.AuthHandler.login_page(conn)
  end

  # Protected pages — auth required
  fn route(conn: Conn(WithSession), :get, ["dashboard" | rest]) do
    case require_auth(conn) do
      Ok(authed_conn) -> MyApp.DashboardRouter.route(authed_conn, :get, rest)
      Error(err_conn) -> err_conn |> redirect("/login")
    end
  end

  # API routes — different content type, token auth
  fn route(conn: Conn(WithSession), method, ["api" | rest]) do
    conn
    |> require_api_token()
    |> MyApp.API.Router.route(method, rest)
  end
end
```

## Default Endpoint Pipeline

The full security-aware pipeline generated by `forge new`:

```march
mod MyApp.Endpoint do
  import Bastion.Middleware
  import Bastion.Security

  fn call(conn) do
    conn
    |> Bastion.Security.SSL.force_ssl()             # HTTPS redirect (prod only)
    |> Bastion.Security.Headers.defaults()           # X-Frame-Options, HSTS, etc.
    |> Bastion.Security.CSP.default_policy()         # Content Security Policy
    |> Bastion.Middleware.max_body_size(1_048_576)   # 1MB request limit
    |> Bastion.Middleware.static("/static", from: "priv/static")
    |> Bastion.Middleware.RequestID.assign_request_id()
    |> Bastion.Middleware.Logger.log_request()
    |> Bastion.Middleware.parse_body()
    |> Bastion.Middleware.load_session(MyApp.Config.session())
    |> Bastion.Security.CSRF.protect()               # CSRF for form submissions
    |> Bastion.Middleware.Depot.with_pool(MyApp.Repo.pool())
    |> MyApp.Router.route()
  end
end
```
